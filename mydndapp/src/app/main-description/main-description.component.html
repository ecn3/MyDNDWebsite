Project Description

My project idea is to create a one stop location for all reference material available to create a player character in D&D 5e. Many veteran players can attest to how long it takes to go through all the available reference material to create a character. Not including the players handbook there are several publications and over a dozen playtest documents available just for character creation.

The general idea is have different sections related race, subrace, class, archetype, feats, spells, and equipment. It would also have be able to be filtered to remove playtest material, only show playtest material, and show all material. In addition it would have an RNG to simulate die rolls for getting a characters stats.

Web Applications

  • Character creator: Would allow for DMs to create NPCs and for for players to create thier characters
  • Compendium of Knowledge Could be used as an online resource for any information that someone can't recall where they found it and doesn't want to sort through over a thousand pages of information
